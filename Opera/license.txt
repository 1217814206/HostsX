安装SimpleU定制版的时候提示无效的安装包。错误代码：45。这是怎么回事啊？
出现这个原因可能是您之前安装过Opera极速版，如果你想继续使用SimpleU定制版可以用7z等软件，右键解压运行。

Ps.另，Nod32都不报毒，金山.360什么的就误报，国内的厂商和国外厂商果然是有很大差距啊。

下载方法：鼠标移动到.exe上面，后方会出现下载，点击下载就好了。
如果下载需要登录说明100次用完了，我会尽快续期，如果您有注册everbox，可以转存到您的文件夹。

为什么我在安装Opera插件的时候会提示不完整？
这是国内网络访问Opera服务器的原因，您可以使用右键--目标另存为来保存插件，然后从本地安装就可以解决此问题。

为什么用代理进入youtube总是弹出什么证书问题，几乎每打开一个视频都会有此问题？
打开Opera的证书管理，在“证书颁发机构”里面导入您的代理软件的证书即可。

为什么论坛和贴吧的签名看不到了？
实在想不出签名有什么看头，所以在Custom.css默认就给过滤掉了大部分的签名，如果只是暂时性的查看，可以手动关闭广告过滤。
如果不想让Custom.css过滤签名可以收动手修改Custom.css文件里的“各种签名过滤”部分。

有些视频播放不正常，网页展开不正常怎么办？
部分视频网站不正常可能是由部分js影响导致的网页问题。比如：CleanPlayer.user.js和DBankLinker.js，删除即可解决。

部分discuz论坛下载附件扩展名为html该怎么办？
(会导致下载站下载以下扩展名文件时不能弹出下载对话框)
[File Types]
text/html=2,,,,rar,zip,7z,apk,sis,sisx,htm,html,shtml,|

SongTaste的歌曲为什么不能正常播放？
用IE打开SongTaste任一播放链接一次，再用opera，打开就能播放了。

5sing的歌曲为什么无法正常播放？
请下载安装微软的silverlight插件，重启Opera后即可正常播放。
https://www.microsoft.com/getsilverlight/get-started/install/default.aspx?reason=unsupportedbrowser

Urlfilter出现uuid了怎么办？
UUID:是Opera官方加入的，如果不喜欢。
可以采用支持正则的文本编辑器进行替换：正则匹配=UUID:\w+$ 替换为空格就行了。

为什么没有64位的Opera SimpleU定制版？
很抱歉，因为我这里没有64位的环境，不会发布64为版本的安装版，您可以下载7z压缩包配置文件，
下载Opera官方64位安装包解压。然后用“Opera SimpleU配置文件.7z”解压后，
覆盖官方的解压版本，即可得到一个64位的Opera SimpleU定制版。

按钮里面涉及到CSS数字勾选的，其中的数字是如何确定的？
Select user CSS file,6 表示使用User CSS文件夹里第6（不过是从0开始数起）个css文件（按文件名升序排序）。
所以这个数字6可能要修改。

为什么我觉得SimpleU定制版占用内存很大？
你把快速拨号全部删除掉，内存占用一下子就掉下来了。

双击Opera图标后，出现Startup error的白底黑字的对话框怎么办？
解压/剪切到其他目录运行试试。

我不想使用自动翻页该如何解决？
在页面载入时按住Ctrl+鼠标左键可以临时关闭。
如果您不想使用此功能可以删除Super_preloader.js Super_preloader.db.js两个js文件解决。

有些页面图片显示不正常，CSS加载不正产或者出现其他诡异的问题怎么办？
清楚持久储存，或者清除所有历史内容试试，或许就能解决了。

为什么我使用SimpleU后，反馈有些网站的广告过滤不完全，其他SimpleU用户却无法重现？
SimpleU定制版需要配合HostsX的数据使用的，才可以获得更好的广告过滤效果。
将部分去广告数据交由系统处理，即加快了Opera的启动速度，也获得了更好的平台兼容性。

为什么我的Opera SimpleU 定制版acid3测试只得了98分？
部分JS可能影响得分情况，请删除OnlinedownNoAds.user.js此文件，即可得满分。

如何取消对cnnic证书的信任？
打开Opera的证书管理，需要手动配置添加此证书的信任排除
文件位于 profile\opcacrt6.dat;profile\opicacrt6.dat

SimpleU定制版为什么没有绿色版？
SimpleU定制版调用了Opera自带的安装界面。
当然，对于想使用绿色版的朋友可以用7-zip等压缩软件右键直接解压到一个干净的目录即可使用。

设置Opera为默认浏览器后，为什么有些SimpleU定制版的功能无法使用了？
如果您要设置Opera为默认浏览器，请运行Opera目录下的 GreenPath.exe 文件，勾选使用绝对路径，以解决相对路径问题！！！

为什么访问有些网站会出现页面显示错乱？
请尝试关闭CSS里面的微软雅黑的显示，看能否可解决此问题。
或者把最小字体由12改为默认的10试试看。

为什么我的Opera滚动时会出现文字出现模糊重影的问题？
请检查是否开启了IDM下载软件监视Opera的选项。关闭即可解决。若未安装IDM，请尝试将此问题提交给官方予以重视。

为什么会有ieopera和ieonly两个调用IE的JS文件？
运行program\OperaJR.exe ，得到reg文件，双击导入。在浏览很多银行的时候便会自动调用IE打开了。

设置 Opera Dragonfly 为离线版或实验版的方法？
 如果在安装时将Opera安装目录更改为“d:\Program Files\Opera”，则可以默认使用本地化DragonFly。
 1.在 https://dragonfly.opera.com/app/stp-1/zips/latest/ （最新稳定版）
      或者 https://dragonfly.opera.com/app/stp-1/experimental/zips/latest/ （最新实验版）
   下载一个 .zip 压缩包（有各个地区语言版本，随你喜欢选一个）。
   最新稳定版 Dragonfly 是针对最新正式版 Opera 使用的，在旧版本 Opera 上使用不保证没有额外的问题。
 2.在硬盘里任意位置解压这个压缩包，把里面的 client-*.xml 拖进浏览器 （文件名因各个地区语言版本而异，总之后缀名是xml那个就错不了） ，然后复制这时地址栏的地址，是以 file://localhost 开头的文件路径。
 3.在 Opera 中打开 opera:config#DeveloperTools，设置值填入刚才得到的文件路径，保存设置。搞定。如果你已经开着 Dragonfly，关掉。

实验版 Dragonfly 的设置方法？
 在 Opera 中打开 opera:config#DeveloperTools，设置值为 https://dragonfly.opera.com/app/experimental/ ，保存设置。搞定。
 
为什么我设置的 UserJS 黑白名单无效？
 // ==UserScript==
 // @include http://xxx.xxx.xxx/*
 // @exclude http://zzz.zzz.zzz/*
 // ==/UserScript==
 其中以 include 标注的即白名单，exclude的为黑名单，可以设置多条。
 若黑白名单失效，请保存脚本为 UTF-8 without BOM 格式。
