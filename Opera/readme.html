常见问题解答：QA

  1. Opera 官方中文社区上不去怎么解决？
    无法正常访问Opera相关网站的同学请注意，请将下列条目加入 hosts 文件
    Windows 下位置为 C:\Windows\System32\drivers\etc/hosts
    Linux/Mac 下为 /etc/hosts
    59.151.106.253 operachina.com
    59.151.106.253 bbs.operachina.com
    59.151.106.253 mini.operachina.com
    59.151.106.253 portal.operachina.com
    59.151.106.253 press.operachina.com
    59.151.106.253 redir.operachina.com
    59.151.106.253 static.operachina.com
    59.151.106.253 wiki.operachina.com
    59.151.106.253 www.operachina.com
    您也可以使用SimpleU自带的HostsTool来更新Hosts文件。
  2. 当您反馈问题时必要的提交出现问题的网址，和相应的重现步骤。否则我解决不了。
    当您反馈说为什么不好用啊，有些网站兼容性不好啊诸如此类的问题我都不会解决的。因为没有网址和重现步骤。
  3. 安装SimpleU定制版的时候提示无效的安装包。错误代码：45。这是怎么回事啊？
    出现这个原因可能是您之前安装过Opera极速版，如果你想继续使用SimpleU定制版可以用7z等软件，右键解压运行。
  4. 下载Opera SimpleU定制版后，金山毒霸和360杀毒，报毒怎么办？
    请本着疑者不用，用者不疑的态度处理此问题。也可以提交到http://www.virscan.org/ 此网站此行检测。optools下有很多第三方脚本和程序，会被误报。
    Ps.Nod32不报毒，金山.360什么的就误报，国内的厂商和国外厂商果然是有很大差距啊。
  5. 为什么我使用SimpleU后，反馈网站的广告过滤不完全，其他SimpleU用户却无法重现？
    SimpleU定制版需要配合HostsX的数据使用的，才可以获得更好的广告过滤效果。将部分去广告数据交由系统处理，既加快了Opera的启动速度，也获得了更好的平台兼容性。
  6. 为什么浏览一些网站的时候会遇到Flash基本上都会卡50%的cpu的情况？
    opera 10.1 opera11.5正式版 firefox都无这个问题，opera next有问题。
    解决方法:opera:config#Performance|SwitchAnimationThrottlingInterval这个选项，不能设置为0,否则出现上面的情况。
  7. 为什么我上某一些网站的时候会很卡，而其他的却很快？
    首先请确定较慢的网站的网址，查看是否具有规律性；
    然后将此网站用IE打开测试是否很卡，比如一些政府网站；
    再次确认网络环境，是否是和网络有关的问题，比如一些墙外网站或当前代理速度不理想；
    或者排查一下是否为系统防护软件或者广告过滤工具的问题，比如XX卫士的XX规则，一些代理调度软件Proxomitron，privoxy等；
    最后，进行过滤排查测试。如果上述仍不能解决问题，请去论坛反馈。
  8. 为什么我在安装Opera插件的时候会提示不完整？
    这是国内网络访问Opera服务器的原因，您可以使用右键--目标另存为来保存插件，然后从本地安装就可以解决此问题。
  9. 我保存了笔记，可是忘记了当时的页面地址，怎么找到？
    如果在网页上保存笔记后，双点笔记后还能打开保存笔记页面的链接。
  10. 页面上有一个Bookmarklets工具，我如何快速保存并应用？
    1.您可以右键添加到收藏夹。
    2.按住Shift可以将任意按钮拖动到任意位置。可以将 Bookmarklets 拖动到任意位置变成按钮方便使用。
  11. 为什么用代理进入youtube总是弹出什么证书问题，几乎每打开一个视频都会有此问题？
    这是由于您使用了代理软件（比如GAE）导致的，打开Opera的证书管理，在"证书颁发机构"里面导入您的代理软件的证书即可。
  12. 为什么我的 Google Reader（谷歌阅读器)不能使用J、K这两个按键上下切换新闻条目了？
    这是由于Simpleu版本为了是google图片的功能正常使用，默认将"www.google.com"的站点首选项识别为Firefox了，
    请在站点首选项-网络- 中 将"浏览器识别"改为"识别为Opera"即可正常在Reader中使用J、K等快捷键。
  13. 为什么论坛和贴吧的签名看不到了？
    实在想不出签名有什么看头，所以在Custom.css默认就给过滤掉了大部分的签名，如果只是暂时性的查看，可以手动关闭广告过滤。
    如果不想让Custom.css过滤签名可以收动手修改Custom.css文件里的"各种签名过滤"部分。
  14. 有些视频播放不正常，网页展开不正常怎么办？
    部分视频网站不正常可能是由部分js影响导致的网页问题。比如：CleanPlayer.user.js和DBankLinker.js，删除即可解决。
  15. 部分discuz论坛下载附件扩展名为html该怎么办？
    (会导致下载站下载以下扩展名文件时不能弹出下载对话框)
    [File Types]
    text/html=2,,,,rar,zip,7z,apk,sis,sisx,htm,html,shtml,|
  16. SongTaste的歌曲为什么不能正常播放？
    用IE打开SongTaste任一播放链接一次，再用opera，打开就能播放了。
  17. 5sing的部分歌曲为什么无法正常播放？
    请下载安装微软的silverlight插件，重启Opera后即可正常播放。
  18. 如何升级Opera？
    1.官方版本的Opera可以使用自动更新，用以更新Opera的版本。
    2.对于定制版的Opera，升级或回滚到旧版可直接覆盖安装。
    对于版本构架变动较大的情况（如Opera10--Opera10.5），不建议沿用旧的配置，最好重新配置用户配置。
    因为新版Opera的安装包是7z格式，所以升级还可以用直接解压缩安装包的方法。
    用压缩工具打开Opera安装文件，除operaprefs_default.ini和files_list外直接选择解压覆盖。
    这种方式升级可以自行选择覆盖更新哪些内容。
  19. Urlfilter出现uuid了怎么办？
    UUID:是Opera官方加入的，如果不喜欢。
    可以采用支持正则的文本编辑器进行替换：正则匹配=UUID:\w+$ 替换为空格就行了。
  20. 为什么没有64位的Opera SimpleU定制版？
    很抱歉，因为我这里没有64位的环境，不会发布64为版本的安装版，您可以下载7z压缩包配置文件，
    下载Opera官方64位安装包解压。然后用"Opera SimpleU配置文件.7z"解压后，
    覆盖官方的解压版本，即可得到一个64位的Opera SimpleU定制版。
  21. 按钮里面涉及到CSS数字勾选的，其中的数字是如何确定的？
    Select user CSS file,6 表示使用User CSS文件夹里第6（不过是从0开始数起）个css文件（按文件名升序排序）。
    所以这个数字6可能要修改。
  22. 为什么我觉得SimpleU定制版占用内存很大？
    你把快速拨号全部删除掉，内存占用一下子就掉下来了。
  23. 双击Opera图标后，出现Startup error的白底黑字的对话框怎么办？
    可能是您的目录权限有问题，重新使用管理员权限进行安装，或者解压/剪切到其他有权限的目录运行试试。
  24. 我不想使用自动翻页该如何解决？
    在页面载入时，按住ctrl键，然后按住鼠标左键，在页面空白处往下拉动鼠标，可以使自动翻页状态暂停！
    如果您不想使用此功能可以删除Super_preloader.js Super_preloader.db.js两个js文件解决。
  25. 有些页面图片显示不正常，CSS加载不正产或者出现其他诡异的问题怎么办？
    清除持久储存，或者清除所有历史内容试试，或许就能解决了。
  26. 为什么我的Opera SimpleU 定制版acid3测试只得了98分？
    部分JS可能影响得分情况，请删除OnlinedownNoAds.user.js此文件，即可得满分。
  27. 如何取消对cnnic证书的信任？
    打开Opera的证书管理，需要手动配置添加此证书的信任排除。
    证书文件位于 profile\opcacrt6.dat;profile\opicacrt6.dat，希望您能及时备份。
  28. SimpleU定制版为什么没有绿色版？
    SimpleU定制版调用了Opera自带的安装界面。 当然，对于想使用绿色版的朋友可以用7-zip等压缩软件右键直接解压到一个干净的目录即可使用。
  29. 设置Opera为默认浏览器后，为什么有些SimpleU定制版的功能无法使用了？
    如果您要设置Opera为默认浏览器，请运行Opera目录下的 GreenPath.exe 文件，勾选使用绝对路径，以解决相对路径问题！！！
  30. 为什么访问有些网站会出现页面显示错乱？
    请尝试关闭CSS里面的微软雅黑的显示，看能否可解决此问题。
    或者把最小字体由12改为默认的10试试看。
  31. 为什么我的Opera滚动时会出现文字出现模糊重影的问题？
    请检查是否开启了IDM下载软件监视Opera的选项。关闭即可解决。若未安装IDM，请尝试将此问题提交给官方予以重视。
  32. 为什么我复制动态图像粘贴到QQ聊天窗口上就是静态的？
    解决方法如下：
    1. 操作错误。应该打开文件发送动态图像，不是复制粘贴。
    2. 除IE内核的浏览器外。其他内核的浏览器均不能复制动态图。即便是复制，也只能复制一帧而已。
  33. 为什么会有ieopera和ieonly两个调用IE的JS文件？
    运行program\OperaJR.exe ，得到reg文件，双击导入。在浏览很多银行的时候便会自动调用IE打开了。（安装版自动包含并省略了此手动步骤。）
  34. 设置 Opera Dragonfly 为离线版或实验版的方法？
    如果在安装时将Opera安装目录更改为"?:\Program Files\Opera"，则可以默认使用本地化DragonFly。
    1.在 https://dragonfly.opera.com/app/stp-1/zips/latest/ （最新稳定版）
    或者 https://dragonfly.opera.com/app/stp-1/experimental/zips/latest/ （最新实验版）
    下载一个 .zip 压缩包（有各个地区语言版本，随你喜欢选一个）。
    最新稳定版 Dragonfly 是针对最新正式版 Opera 使用的，在旧版本 Opera 上使用不保证没有额外的问题。
    2.在硬盘里任意位置解压这个压缩包，把里面的 client-*.xml 拖进浏览器 （文件名因各个地区语言版本而异，总之后缀名是xml那个就错不了） ，然后复制这时地址栏的地址，是以 file://localhost 开头的文件路径。
    3.在 Opera 中打开 opera:config#DeveloperTools，设置值填入刚才得到的文件路径，保存设置。搞定。如果你已经开着 Dragonfly，关掉并重新打开即可。
  35. 实验版 Dragonfly 的设置方法？
    在 Opera 中打开 opera:config#DeveloperTools，设置值为 https://dragonfly.opera.com/app/experimental/ ，保存设置。搞定。
  36. 为什么我设置的 UserJS 黑白名单无效？
    // ==UserScript==
    // @include http://xxx.xxx.xxx/*
    // @exclude http://zzz.zzz.zzz/*
    // ==/UserScript==
    其中以 include 标注的即白名单，exclude的为黑名单，可以设置多条。
    若黑白名单失效，请保存脚本为 UTF-8 without BOM 格式。
  37. 为什么有时候菜单的参数无效呢？
    Opera有自己的定义函数：
    %u = 当前显示文档的url，常用于下载工具传递本页所有链接
    %t = 选定文本。比如常用的"转到URL"的命令为Go to page, "%t"
    %s = 来源于当前显示文档的字符串
    %c = 剪贴版内容
    %i = 显示的搜索结果个数 (search.ini)
    %l = 当前选择的链接，常用于下载工具传递链接
  38. 我用的Opera12版本，怎么感觉浏览器变慢了？
    Opera12补充优化：
    1、开启硬加速：0禁用、1自动、2强制开启
    在开启两种渲染器的情况下，如需更换不同的渲染器请把opera:config#Preferred renderer设置：0为OpenGL 、1为DirectX
    Enable Hardware Acceleration：opera:config#UserPrefs|EnableHardwareAcceleration
    Enable WebGL：opera:config#UserPrefs|EnableWebGL
    2、网络缓存Network Buffer Size设置1，有些网页可能会出问题，可以逐步调大点参数。设置为1对网页加快载入效果较为明显。
    注意，如果存在两个版本的FLASH,把32或64版本其中之一关掉。
    opera:config#Performance|NetworkBufferSize设置为1，设置12可能是个比较合适的参数。
    3、动画性能优化opera:config#Performance|Animation，把他们设置为0，这个设置有些时候会导致加载Animation更慢，大家酌情处理，但设置为0时，加载缓存会降低延迟。
    opera:config#Performance|LagThresholdForAnimationThrottling
    opera:config#Performance|SwitchAnimationThrottlingInterval
  39. 部分版本备注：
    Opera 从 11.60版开始，对CSS的选择器区分大小写。
    11.00以下,display:none元素不下载,9.64表现尤其好...
    Opera 11.10以后新增几个延迟选项，微调以下选项可以提升页面响应速度（要最快就设置全为0）
    1. opera:config#Styled First Update Timeout
    2. opera:config#First Update Delay
    3. opera:config#Update Delay
    4. opera:config#Wait For Style
    5. opera:config#Reflow Delay Load
    6. opera:config#Reflow Delay Script
